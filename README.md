# ⛵ Lifeboat - Personal Life Management Application

A modern, fast, and beautiful, privacy-first local personal life management application built with Python and CustomTkinter.

## Quick Start

Clone the repository and install dependencies:

```bash
#Either clone the repository or download the source code
git clone
cd LifeBoat
pip install -r requirements.txt
```

To Execute the application, run:

```bash
# Run the application
python main.py

# Or use the script
cd scripts
run.bat
```

## Features

- 📊 **Dashboard** - Overview of all your data
- 📅 **Calendar** - Event management and scheduling
- ✓ **Tasks** - Advanced task management
- ☑ **Todo** - Simple daily todos
- 💰 **Expenses** - Track income and expenses
- 🎯 **Goals** - Set and track your goals
- 📝 **Notes** - Take and organize notes
- 🔄 **Habits** - Build and track habits
- ⚙ **Settings** - Customize themes and preferences
