## 📦 Installation

### Standard Installation

Download and run `LifeBoat_Standard_Installer.exe` for a traditional installation with system integration.

### Portable Version

Download and run `LifeBoat_Portable_Installer.exe` to create a portable version that can run from any location.

### From Source

```bash
git clone https://github.com/yourusername/lifeboat.git
cd lifeboat
pip install -r requirements.txt
python main.py
```
