# Lifeboat v2.7.4 - Bug Fix Release

**Release Date:** February 28, 2026  
**Build Type:** Beta  
**Tag:** `v2.7.4`

---

## 🐛 Critical Bug Fixes

This release focuses on fixing critical bugs and improving stability.

### Fixed Issues

1. **Calendar Month Navigation Crash** - Fixed crash when navigating months on dates 29, 30, or 31
2. **Sidebar Navigation Desync** - Sidebar now correctly highlights the active section
3. **Silent Error Handling** - Replaced silent failures with proper error messages across all features
4. **Todo Date Calculation** - Fixed crash in todo date handling

## 🔄 Update System Improvements

- Implemented lightweight GitHub releases checker
- Improved update dialog UX with better styling and animations
- All dialogs now have properly rounded corners

## 📦 Installation

Download the installer for your preferred installation type:

- **Standard Installer**: Installs to Program Files (requires admin)
- **Portable Installer**: Installs to user directory (no admin required)

## 🔧 Technical Changes

- Custom exception classes for better error handling
- Simplified update checking system
- Global dialog styling improvements
- Reduced codebase size and complexity
